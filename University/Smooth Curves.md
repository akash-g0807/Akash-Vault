# Smooth Curve
---
```ad-Definition

Title: **Smooth Curve**

An [[Oriented Curves|oriented curve]] C is **smooth** if there exists a [[Oriented Curves#^a5558f|parametrization]] $\underline{p}$ such that
$$\underline{\dot{p}} \neq 0 \ \ \ \ \ \ \ \forall t$$
```
